Keep Track of My Games
Web App, Non-Commercial
KTOMG is a non-commercial app that lets you manage your game collection and track new releases. I run, host, and manage it by myself. It's developed in .NET, hosted in a hybrid IaaS/PaaS environment in Azure and runs on the RavenDB document database.
Excalibur.js
Game Engine, Open Source
Excalibur is an open source HTML5 game engine written in TypeScript. I participate as one of the core contributors and contribute to CI/CD automation, API design, documentation, examples, and other engine features (except physics, that's Erik's specialty).

Bowling Calculator

Windows Phone App, Open Source
Bowling Calculator is a free Windows Phone app that helps you calculate scores during a bowling game. I developed it as an open source project to showcase localization (English/French) and MVVM patterns for Windows Phone apps.

Centwise

Retired • Windows Phone App
Centwise was a Splitwise client for Windows Phone. It was written in XAML and C# as a native app. I maintained it for several years as a paid app, then switched to free, and finally retired it after adoption was too low to make it worth supporting the new API versions.
