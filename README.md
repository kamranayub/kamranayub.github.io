## Kamran Ayub Personal Blog

- Built with Wyam static site generator
- Semantic UI framework

### Build

**Before**

Install Wyam so it's available on PATH.

**In Visual Studio Code:**

- Ctrl-Shift-B: build all
- Ctrl-P, `task`, then choose task

**Command-line**

    # Build all
    npm run build

    # Build Semantic UI
    npm run semantic

    # Watch
    npm start

    # Preview
    wyam preview
